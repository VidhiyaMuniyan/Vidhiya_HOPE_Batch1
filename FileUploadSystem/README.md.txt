#File Upload System (Java)

##  Description
This project is a simple File Upload System developed using Object-Oriented Programming (OOP) concepts in Java.

##  Features
- Upload Image Files
- Upload Document Files
- Storage Management
- File Size Validation
- Error Handling

##  OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

##  Project Structure
- File.java (Abstract Class)
- ImageFile.java
- DocumentFile.java
- User.java
- StorageManager.java
- FileService.java
- Main.java
