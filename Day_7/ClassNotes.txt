Inheritance:
  Properties / behaviours that's been transferred from parent class to child class is called inheritance.
  Child class uses the features of parent class.

What is the hierarchy of compilation:
 mainclass will be compiled and main class file will be generated.

Types of Inheritance:
  1. Single 
  2. Multilevel
  3. Multiple
  4. Heirachical
  5. Hybrid

1,2,4 - these 3 types of inheritance directly supports inheritance.
3 and 5 - indirectly.

This keyword:
  Purpose of this keyword is to use the current object memory addresses/values.
  To cross verify the purpose of this keyword you can utilize hashcode to see its memory allocation.

Super() keyword: Used to access the parent class members.
